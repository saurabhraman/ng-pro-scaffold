# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-03

### Added

- Initial release of ng-pro-scaffold CLI tool
- Interactive project scaffolding with Inquirer.js prompts
- Non-interactive mode support for CI/CD pipelines
- Angular project generation with modern best practices
- Optional Tailwind CSS integration
- Optional Docker configuration with multi-stage builds
- Optional Angular Material component library integration
- Cross-platform support (Windows, macOS, Linux)
- Package manager selection (npm, yarn, pnpm)
- Security-first approach with input sanitization
- Comprehensive test suite with Jest
- ESLint configuration for code quality
- Prettier integration for code formatting
- Husky for git hooks

### Features

- 🚀 Quick setup - Generate production-ready Angular projects in seconds
- 🎨 Tailwind CSS - Optional utility-first CSS framework
- 🐳 Docker Ready - Optional containerization with nginx
- 📦 Angular Material - Optional component library
- 🔒 Security First - Input sanitization and secure execution
- 🌍 Cross-Platform - Works on all major operating systems

### Templates

- Base Angular project structure
- Docker configuration files (Dockerfile, docker-compose.yml, nginx.conf)
- TypeScript configuration files
- Angular CLI compatible project structure
- Home component as starting point
